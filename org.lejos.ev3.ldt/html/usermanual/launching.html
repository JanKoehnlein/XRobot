Uploading a leJOS EV3 program to the EV3 brick

To link and upload a program to the EV3 brick, first make sure that the project is a LeJOS EV3 project by checking for a little LeJOS icon next to the project name. If the project is not a LeJOS EV3 project, or you haven't created a project yet, follow these instructions. Once a LeJOS EV3 project is set up, follow these steps:

  1. In the Package Explorer of the Java perspective, right click on the project, a package, a *.java file or a class
  2. Select Run As→LeJOS EV3 Program
  3. Eclipse might ask you to select the main class, if multiple suitable classes exist in your selection

Note; if some error occurs, consult Eclipse's Console view for details.
