Configuration

Configuration for the Plugin takes place in the leJOS EV3 section of the Preferences page of Eclipse.
Navigate there by calling Window→Preferences... from the menu bar of the Eclipse workbench.

Setting EV3_HOME

You need to set the property EV3_HOME in the preference page of leJOS EV3. Click the Browse button that is located right of the according text field and select the root directory of your leJOS EV3 installation in the file system.

Running the program immediately after the download

You can choose to run your leJOS program immediately after it has been downloaded by checking the Run program after download box.
