Welcome to XRobots

About XRobots

XRobots is a fighting game that involves two Lego Mindstorms robots. Players program the robots on-line using a scripting language based on Xtext. The scripts can be tested in live matches at various events.

Getting Started

  • Bookmark this page: it's a permanent link to your XRobots script. You can save your script and come back to it anytime later.
  • Edit the script as you like. Use ctrl+space to get content assistance at the current cursor position.
  • Find general documentation on the Documentation page and an API description including available commands on the JavaDoc page above.
  • Go to the itemis booth, choose one of the free slots shown on the screen, and enter the token into the execute dialog.
  • Watch the fight and observe you robot's behavior. You may improve your script and repeat the fight as often as you like.
